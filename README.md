🏏 IPL Dashboard Analysis – Excel Project


I created an interactive IPL Dashboard in Excel to analyze Indian Premier League match data and convert raw cricket data into useful insights.
The main objective of this project was to analyze:

-> Team performance
-> Player performance
-> Match results
-> Runs and wickets analysis
-> Season-wise trends

I used Excel features like Pivot Tables, Pivot Charts, Slicers, KPI Cards, and Conditional Formatting to build the dashboard.
This dashboard gives all IPL insights in one place and helps users quickly understand overall tournament performance.

📊 2. KPI Used in Dashboard

✅ Total Matches
Shows total IPL matches included in the dashboard.

Use: Helps understand total tournament coverage.

✅ Total Teams
Displays number of teams participating.

Use: Easy team comparison.

✅ Total Runs
Shows total runs scored in all matches.

Use: Helps analyze batting performance.

✅ Total Wickets
Shows wickets taken across the tournament.

Use: Helps evaluate bowling performance.

✅ Highest Team Score
Shows the highest runs scored by a team.

Use: Highlights best batting innings.

✅ Most Runs by Player
Displays top run scorer.

Use: Identifies best batsman.

✅ Most Wickets by Player
Displays highest wicket taker.

Use: Identifies best bowler.

✅ Win Percentage
Shows team-wise winning percentage.

Use: Helps compare consistency.

✅ Toss Win vs Match Win
Shows whether toss impacts match result.

Use: Helps analyze match strategy.
